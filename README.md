# 🔍 Solar Cell Defect Detection using Deep Learning

## 📖 Overview

This project automates the detection of defects in solar cell images using Deep Learning techniques.

The model classifies solar cell images into defective and non-defective categories using Convolutional Neural Networks (CNNs).

---

## 🚀 Features

- Image preprocessing
- Data augmentation
- CNN model training
- Image classification
- Model evaluation
- Defect prediction
